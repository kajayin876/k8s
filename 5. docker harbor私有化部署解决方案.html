1.Harbor 介绍
Harbor 是一个开源可信的云原生注册表项目，用于存储、签名和扫描内容。Harbor 通过添加用户通常需要的功能（如安全性、身份和管理）来扩展开源 Docker 发行版。使注册表更靠近生成和运行环境可以提高映像传输效率。Harbor 支持在注册表之间复制映像，并提供高级安全功能，例如用户管理、访问控制和活动审核。
Harbor由云原生计算基金会（CNCF）托管。如果您是一个希望帮助塑造云原生技术发展的组织，请考虑加入 CNCF。有关谁参与以及Harbor如何发挥作用的详细信息，

特征

  • 云原生注册表：Harbor 同时支持容器镜像和  Helm  图表，可用作容器运行时和编排平台等云原生环境的注册表。
  • 基于角色的访问控制：用户通过“项目”访问不同的存储库，用户可以对项目下的图像或 Helm 图表具有不同的权限。
  • 基于策略的复制：可以使用筛选器（存储库、标记和标签）基于策略在多个注册表实例之间复制（同步）映像和图表。如果遇到任何错误，Harbor 会自动重试复制。这可用于帮助负载平衡、实现高可用性，并促进混合和多云方案中的多数据中心部署。
  • 漏洞扫描：Harbor 会定期扫描镜像中的漏洞，并具有策略检查功能，以防止部署易受攻击的镜像。
  • LDAP/AD 支持：Harbor 与现有的企业 LDAP/AD 集成，用于用户身份验证和管理，并支持将 LDAP 组导入 Harbor，然后可以授予对特定项目的权限。
  • OIDC 支持：Harbor 利用 OpenID Connect （OIDC） 来验证由外部授权服务器或身份提供商进行身份验证的用户的身份。可以启用单点登录以登录到 Harbor 门户。
  • 映像删除和垃圾回收：系统管理员可以运行垃圾回收作业，以便可以删除图像（悬空清单和未引用的 blob），并可以定期释放其空间。
  • 公证：支持使用 Docker 内容信任（利用公证）对容器镜像进行签名，以保证真实性和出处。此外，还可以激活阻止部署未签名映像的策略。
  • 图形化用户门户：用户可以轻松浏览、搜索存储库和管理项目。
  • 审核：通过日志跟踪对存储库的所有操作。
  • RESTful API：提供  RESTful API  是为了方便管理操作，并且易于使用与外部系统集成。嵌入式 Swagger UI 可用于探索和测试 API。
  • 易于部署：Harbor可以通过Docker组合以及Helm Chart进行部署，并且最近还添加了Harbor Operator。

3.创建spring boot 项目并且上传私有harbor仓库
     3.1 java下载
  • wget https://download.oracle.com/java/17/latest/jdk-17_linux-x64_bin.tar.gz  （下载jdk17因为项目是spring boot3.0）
    3.2 解压缩    
  •       tar xf jdk-17_linux-x64_bin.tar.gz   
    3.3 环境变量配置
  •       mv jdk-17.0.8/ /usr/lib/jvm (放到统一目录)
  •        sudo vi /etc/profile 修改java配置
                  export JAVA_HOME=/usr/lib/jvm/jdk-17.0.8
                  export CLASSPATH=$JAVA_HOME/lib:$JRE_HOME/lib:$CLASSPATH
                  export PATH=$JAVA_HOME/bin:$JRE_HOME/bin:$PATH
  •       source /etc/profile （刷新配置）
      
3.2 maven 安装
  • wget  https://dlcdn.apache.org/maven/maven-3/3.9.4/binaries/apache-maven-3.9.4-bin.tar.gz
  • tar -xvf apache-maven-3.9.4-bin.tar.gz
  • sudo mv -f apache-maven-3.9.4 /usr/local/
  • 编辑  /etc/profile  文件  sudo vim /etc/profile，在文件末尾添加如下代码：
  • export MAVEN_HOME=/usr/local/apache-maven-3.9.4
  • export PATH=${PATH}:${MAVEN_HOME}/bin
  •   保存文件，并运行如下命令使环境变量生效：
  • source /etc/profile
  • 在控制台输入如下命令，如果能看到 Maven 相关版本信息，则说明 Maven 已经安装成功：
  • mvn -v
3.3 spring boot Docker file配置
            3.3.1 spring boot 代码地址 https://gitee.com/erp-open/kubernetes  目录下spring-boot-hello
