2.安装virtualbox
  • 先安装microsoft visual c++ 2019
  • 安装virualbox工具
  • 建议virtualbox不要安装c盘，不能占c盘资源
  • 安装完启动

  • 安装直接运行


  • 点击新建把centos iso镜像导入进去，教程是下载的Minimal，

  • 建议文件夹不要放系统盘，因为虚拟机文件都很大

  • 用户名和密码可以自定义
  • 教程内存设置4cpu和8G内存 8G内存是1024*8


  • 点击启动
  • 选择英文
  • 设置密码

  • 大概等20分钟就可以装好centos

  • 网络设置
  • 注意：网卡1和网卡2都需要设置

  • yum update -y 先更新源 -y意思不需要询问直接更新
  • 安装网络 yum install  net-tools package -y
  • yum install -y yum-utils
  • yum-config-manager --add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
